1) How do you display unordered list as square shape?
answer:  use "list-style-type" css property in style attribute of ul element with value "square"
example:
<ul style="list-style-type:square;">
    <li>India</li>
    <li>Australia</li>
    <li>Pakistan</li>
</ul>

2) How do you start the ordered list from 3?
answer: start attribute of ol element and value to 3
example:
<ol start="3">
    <li>India</li>
    <li>Australia</li>
    <li>Pakistan</li>
</ol>
